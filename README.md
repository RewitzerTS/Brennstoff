# Hausgemachte Chili-Saucen Website

Erster statischer Entwurf für GitHub Pages.

## Dateien

- `index.html` – komplette Website mit CSS und JavaScript
- `assets/` – deine Label-Bilder

## Veröffentlichung auf GitHub Pages

1. Neues Repository erstellen, z. B. `chili-saucen-website`.
2. Alle Dateien aus diesem Ordner hochladen.
3. In GitHub: **Settings → Pages** öffnen.
4. Unter **Build and deployment** die Quelle **Deploy from a branch** wählen.
5. Branch `main` und Ordner `/root` auswählen.
6. Speichern.

## Nächste Anpassungen

- In `checkoutByMail()` die Adresse `deine-mail@example.com` ersetzen.
- Preise und Produkttexte in der `products`-Liste anpassen.
- Für echten Verkauf: Impressum, Datenschutz, Widerruf, Versandinfos und Zahlungs-/Bestelllösung ergänzen.
